# Lyrics Overlay Extension - 버전 관리 이력 (CHANGELOG)

이 파일은 확장프로그램의 버전별 변경 사항을 기록합니다.

## [v1.4.1] - 2026-06-21

### 🛠 개선
- **도움말 페이지에 "이전/다음 가사 표시" 기능 설명 추가** (ko/en/ja), 디자인 탭 항목도 최신 라벨("텍스트 그림자"/"외곽선 굵기")에 맞춰 갱신.
- **디자인 설정 내보내기/가져오기 보정**: `siteStates`에서 디자인과 무관한 `currentTrack`(재생 복원용, srtText 포함)을 제외하고 내보내며, 가져올 때는 기존 `currentTrack`을 보존하도록 수정.
- **설정 데이터 내보내기 보정**: 한 번도 저장된 적 없는(기본값 그대로인) 설정도 누락 없이 내보내지도록 수정.
- **도움말 페이지 표기 정리**: 사이트별 테마 색상에서 "치지직 / 기본" 표기를 한국어는 "기본 (치지직)"으로, 영어/일본어는 "Default"/"デフォルト"만 남기도록 정리. 목록 항목 끝 마침표 표기를 통일.

## [v1.4.0] - 2026-06-21

### 🚀 추가된 기능
- **이전/다음 가사 표시 시 가사창 높이 고정**: 곡 내 최대 줄 수를 기준으로 메인/컨텍스트 영역 높이를 미리 확보해, 줄 수가 바뀌거나 빈 구간이 와도 가사창 높이가 출렁이지 않습니다.

### 🛠 개선
- **이전/다음 가사 표시 시 깜빡임 제거**: 줄이 바뀔 때마다 가사창이 닫혔다 열리는 것처럼 보이던 현상을 해결. 박스가 이미 보이는 상태라면 진입 애니메이션을 재생하지 않고 내용만 갱신합니다.
- **빈 구간(공백) 기준 3초 → 5초로 변경**: 더 짧은 간주 구간에서는 직전 가사를 유지합니다.
- **페이드/슬라이드 애니메이션 구분 강화**: 페이드는 순수 투명도 변화로, 슬라이드는 가로 이동으로 시각적으로 명확히 구분되도록 수정.
- **이전/다음 가사 표시 시 애니메이션 옵션 비활성화**: 해당 모드에서는 줄 전환 애니메이션이 거의 재생되지 않으므로 설정창에서 선택을 막고 안내 문구를 표시합니다.
- **"텍스트 외곽선" 라벨 명칭 구분**: 그림자 효과(textShadow)와 굵기 효과(outlineWidth)가 모두 "외곽선"으로 표시되어 혼동되던 것을 "텍스트 그림자"로 구분.
- **Auto Only 가사는 번호 채번 제외**: 숨겨진 Auto Only 항목이 번호를 소비해 일반 곡 번호에 구멍이 생기던 문제 해결.

### 🐛 버그 수정
- **좌/우 정렬 시 위치 저장 오류 수정**: 가사 길이(가사창 너비)가 바뀌면 좌/우 정렬 기준 위치가 중앙 기준으로 환산되어 화면 새로고침/설정 변경 시 위치가 틀어지던 문제 해결. 이제 좌측 정렬은 좌측 끝, 우측 정렬은 우측 끝을 기준으로 위치가 고정됩니다.
- **리모컨 라이브러리에서 곡 재생 시 가사창 높이 고정 미적용 수정**: `playFromRemoteLibrary` 경로에서 높이 고정 재계산이 누락되어 있던 문제 해결.

## [v1.3.3] - 2026-06-13

### 🚀 추가된 기능
- **영상 일시정지/재생 연동 (치지직 VOD · 유튜브)**: 영상을 멈추면 가사도 멈추고, 다시 재생하면 가사도 이어서 재생됩니다. (VOD에서만 동작, 라이브 제외)
- **완전 투명 배경**: 배경 투명도 0% + 블러 0px로 설정하면 테두리·그림자까지 제거되어 가사 텍스트만 표시됩니다.
- **시트 URL 저장 시 즉시 동기화**: 설정에서 구글 시트 URL을 새로 추가하거나 변경한 뒤 저장하면 자동으로 한 번 동기화합니다.
- **리모컨 패널 자동 닫힘**: 가사 목록/타임라인 패널을 열어둔 채 일정 시간 조작이 없으면 자동으로 닫힙니다. (설정에서 on/off 및 시간 지정 가능, 기본 5초)

### 🛠 개선
- **곡 전환 시 타임라인 스크롤 초기화**: 다음 곡으로 바뀌면 리모컨 타임라인 스크롤이 맨 위로 올라갑니다.
- 설정의 자동 닫힘 시간 입력칸을 테마 색상·라운드 스타일에 맞춰 정리했습니다.

## [v1.3.2] - 2026-06-13

### 🛠 가사창 위치/드래그 버그 수정
- **드래그 시작 시 위로 튀는 현상 수정**: 박스가 `bottom` 기준으로 배치되는데 일부 경로가 `top` 기준으로 처리해 박스 높이만큼 위로 솟구치던 문제를 해결. 드래그를 포인터 이동량(delta) 기반으로 변경해 잡는 순간 위치가 그대로 유지됩니다.
- **페이지 고정(Pin) 위치 저장 오류 수정**: 새로고침 시 고정한 가사창이 위로 올라가던 문제 수정. 위치 기준을 '박스 바닥'으로 통일했습니다.
- **중앙 정렬 시 좌우로 흔들리는 현상 수정**: 가사가 바뀌어 너비가 변할 때 중심이 드리프트하던 문제를, 내용 변경 후 중심 재고정으로 해결. 너비 측정을 애니메이션 변형에 영향받지 않는 방식으로 변경했습니다.
- **"Extension context invalidated" 에러 방어**: 확장프로그램 업데이트 후 기존 탭에서 리모컨 조작 시 콘솔 에러가 발생하던 문제에 안전장치를 추가했습니다.

## [v1.3.1] - 2026-06-01

### 🎵 SoundCloud 자동 감지 및 동기화
- **곡 자동 감지**: SoundCloud 재생 중인 곡을 자동으로 감지해 라이브러리에서 매칭되는 가사를 자동 로드
- **재생 위치 싱크**: SoundCloud 플레이어의 실제 재생 위치(재생바)에 맞춰 가사가 자동 동기화
- **SoundCloud 테마**: 리모컨이 SoundCloud 브랜드 색상(#FF5500)으로 자동 변경
- **반복 재생·타임라인 스크럽 대응**: 곡이 반복되거나 플레이어 타임라인을 이동해도 가사가 따라옴
- **성능 최적화**: 라이브러리 메모리 캐시 + DOM 요소 캐시(2초 갱신) + 메타 키 캐시로 불필요한 storage 읽기 및 DOM 쿼리 제거

### 📋 Auto Only 가사
- **구글 시트 D열** `Auto Only`: 값이 `1`이면 해당 가사는 라이브러리 목록·검색·곡 수에서 숨겨지고 자동 감지로만 로드
- **시트 연동 완료 메시지**: 일반 곡과 자동 전용 곡 수를 분리 표시 — `+22 (+1 자동) / ±0`
- **리모컨 자동 연동**: 목록이 비어있을 때 리모컨에서 연동 시 완료 메시지 동일 형식으로 표시

### 📌 가사창 페이지 고정 (Pin)
- **핀 버튼**: 가사창에 마우스를 올리면 우상단에 핀 아이콘이 나타나며, 클릭하면 가사창을 페이지 좌표에 고정 (스크롤해도 움직이지 않음)
- **핀 색상 설정**: 디자인 탭에서 핀 아이콘 색상 변경 가능

### ↔ 가사창 정렬 방향
- 가사창 내부 텍스트 및 박스 자체의 앵커를 왼쪽 / 가운데 / 오른쪽으로 설정 가능

### ⚙️ 팝업 UI 개편
- **디자인 탭 신설**: 기존 설정 탭을 일반 설정(동작·연동)과 디자인(시각) 탭으로 분리
- **사이트별 디자인 개별 설정**: 특정 사이트에만 다른 폰트 크기·색상 등을 적용할 수 있는 덮어쓰기 기능
- **백업 이원화**: 일반 설정 데이터 / 디자인 데이터를 각각 독립적으로 내보내기·가져오기
- **탭별 현재 가사 상태 동기화**: 여러 탭에서 동작 시 팝업이 현재 활성 탭의 가사 정보를 정확히 표시

### 🌐 다국어 지원 확장
- SoundCloud 테마, Auto Only, 핀 고정, 정렬 옵션 등 신규 기능 전체 한국어·영어·일본어 지원

### 🛠 버그 수정
- 핀 모드에서 가사창 위치가 이전 저장 좌표로 튀는 버그 수정
- 핀 모드에서 영역 선택 시 가사창이 사라지는 버그 수정

## [v1.1.2] - 2026-05-08

### 🚀 추가된 기능
- **검색 매치 하이라이트**
  - 라이브러리 검색 시 각 곡이 어떤 필드(가수, 제목, 키워드 등)에 걸려 검색됐는지 칩(chip) 형태로 표시됩니다.
  - 매치된 글자는 강조색으로 하이라이트되어 시각적으로 확인 가능합니다.
  - 리모컨의 노래 목록 검색에서도 동일하게 동작합니다.
- **구글 시트 자동 연동**
  - 라이브러리가 비어있고 구글 시트 URL이 설정된 경우, 확장프로그램 팝업을 열거나 리모컨 노래 목록을 열 때 자동으로 시트에서 데이터를 불러옵니다.
  - 리모컨에서 자동 연동 시 진행 스피너와 완료 메시지를 표시합니다.
- **시트 연동 진행 상태 표시**
  - 구글 시트 동기화 중 팝업의 시트 연동 버튼이 "연동 중..."으로 변경되며 완료 후 복원됩니다.
  - 중복 클릭이 방지됩니다.

### 🛠 개선
- **라이브러리 목록 정리**: 각 곡 항목에서 등록 날짜 표시를 제거하여 UI를 간결하게 개선했습니다.
- **내부 구조 개선**: CSV 파싱 및 파일명 파싱 로직을 `lib/sheet-parser.js`로 분리하여 팝업과 리모컨 간 코드를 공유합니다.

## [v1.1.0] - 2026-05-04 (배포 버전)

### 🚀 추가된 기능
- **플로팅 리모컨 신규 추가 (Remote Control)**
  - 확장프로그램 팝업을 열지 않고도 화면 위에서 바로 가사 선택, 재생 제어, 싱크 조절이 가능한 컨트롤러가 추가되었습니다.
  - **상시 표시 (사이트별 설정)**: '리모컨 ON' 버튼을 누르면 해당 사이트(도메인) 방문 시 리모컨이 항상 표시되어 즉시 가사를 검색하고 띄울 수 있습니다.
  - **자유로운 배치**: 드래그하여 화면 어디든 원하는 위치에 배치할 수 있으며, 영상 시청에 방해가 되지 않도록 작게 접어두는 **최소화 기능**을 제공합니다.
  - **스마트 패널**: 리모컨의 위치(화면 상단/하단)를 자동으로 감지하여 가사 목록 및 타임라인 패널이 가려지지 않는 방향으로 똑똑하게 펼쳐집니다.
  - **플랫폼 맞춤 디자인**: 접속 중인 플랫폼에 맞춰 테마 색상이 자동으로 변경되어 이질감 없는 디자인을 제공합니다 (기본: 주황, 치지직: 초록, 유튜브: 빨강).
  - **개별 버튼 커스텀**: 설정 탭에서 리모컨에 표시할 버튼(목록, 타임라인, 재생/정지, 싱크)을 사용자 취향에 맞게 선택하여 구성할 수 있습니다.
- **텍스트 자동 스크롤 (Marquee Effect)**
  - 플레이어 탭 및 라이브러리 탭에서 곡 제목/가수명이 길 경우 자동으로 스크롤되어 표시.
- **곡 종료 알림**
  - 가사가 모두 끝난 뒤 1초 후에 `🎵 Finish 🎵` 문구를 화면에 표시 (설정 탭에서 켜고 끄기 가능).
- **구글 시트 연동 개선**
  - C열(세 번째 열)에 입력된 텍스트를 검색 추가 키워드로 파싱하도록 개선.
- **백업/복원 기능 강화**
  - 기존 설정 및 라이브러리 백업 시스템에 리모컨 사이트별 ON/OFF 및 화면상 위치 정보까지 모두 포함되도록 일원화.
- **About(정보) 섹션**
  - 설정 탭 하단에 제작자 정보 및 AI 제작 지원(Gemini, Claude) 문구 추가.

- **점프 감지 로직 고도화**: 타임라인 클릭(Seek) 시 가사 싱크가 꼬이거나 불필요한 알림이 뜨는 현상을 방지.
