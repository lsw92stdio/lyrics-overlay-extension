1 기본적인 사용 방법
🔘 팝업 헤더 — 리모컨 ON/OFF

확장프로그램 아이콘을 클릭하면 팝업 상단에 리모컨 ON / 리모컨 OFF 버튼이 있습니다. 이 버튼은 현재 열린 사이트에만 적용되는 사이트별 설정으로, ON 상태일 때 해당 페이지 위에 플로팅 리모컨이 표시됩니다.

🎵 플레이어 탭
  • SRT 로드: 라이브러리에서 곡을 클릭하거나 플레이어 영역에 .srt 파일 드래그 앤 드롭
  • 재생 / 일시정지 (▶ / ⏸): 가사 진행 제어
  • 정지 (■): 재생을 완전히 멈추고 처음으로 돌아감
  • 프로그레스 바: 클릭하면 해당 위치로 바로 이동
  • 타임라인: SRT 로드 후 플레이어 탭 하단에 가사 목록이 표시됩니다. 각 항목을 클릭하면 해당 가사 위치로 이동합니다.
  • 싱크 조절: 영상과 가사 타이밍이 안 맞을 때 조정
    ← → ±100ms  |  Shift+←→ ±500ms  |  R 또는 UI의 ↺ 버튼으로 초기화
📁 라이브러리 탭
  • 파일 추가: 탭 안으로 .srt 파일을 드래그 앤 드롭하면 여러 파일을 한 번에 등록 가능
  • 검색: 번호·가수·제목·키워드로 검색. 매칭된 항목은 어떤 필드에서 검색됐는지 칩(chip)으로 표시됨
  • 시트 연동: 설정에 저장된 구글 시트 URL에서 데이터를 가져와 라이브러리에 병합
  • 시트 양식 복사: 현재 저장된 가사 전체를 구글 시트에 붙여넣을 수 있는 형식(TSV, 3열)으로 클립보드에 복사
  • 일괄 삭제: 저장된 모든 가사 삭제 (확인 대화창 표시)
  • 정보 관리: 제목·가수(원문/한글)·번호·키워드·Auto Only·영상 동기화 등 메타데이터를 한 화면에서 일괄 조회/수정하는 별도 페이지를 새 탭으로 엽니다
  • 자막 수정: 정보 관리 페이지의 각 항목에서 ✎ 버튼으로 자막 타이밍 일괄 조정, 줄 단위(원문/발음/해석) 수정, 텍스트 복사/붙여넣기가 가능한 편집창을 팝업으로 엽니다
⚙️ 설정 탭 (일반)
  • 곡 종료 알림 마지막 가사 이후 🎵 Finish 🎵 알림 표시
  • 진행 바 브라우저 하단에 재생 위치 표시 바 표시 여부
  • 노래 자동 감지 "관리" 버튼을 누르면 뜨는 창에서 SoundCloud / YouTube / YouTube Music을 각각 켜고 끌 수 있습니다. 재생 중인 곡을 자동으로 감지해 가사를 로드하고 동기화합니다
  • 리모컨 버튼 플로팅 리모컨에 표시할 버튼 개별 선택 (라이브러리·타임라인·영역 선택·재생/정지·싱크 조절). URL 동기화 버튼(🔗/🔓) 표시 여부도 별도로 켜고 끌 수 있습니다
  • 가수/제목 표시 라이브러리 및 플레이어에서 이름 표시 방식: 원문+한국어 / 원문 우선 / 한국어 우선
  • 구글 시트 URL 시트 연동에 사용할 스프레드시트 주소
  • 이전 가사 표시 / 다음 가사 표시 현재 가사 위·아래에 이전·다음 줄을 작게 함께 표시합니다(노래방 화면처럼)
🎨 디자인 탭
  • 원문 / 발음 / 한국어 각 줄의 표시 여부, 폰트 크기, 색상 개별 설정
  • 배경 배경색, 투명도(0~100%), 블러 강도
  • 애니메이션 페이드 / 슬라이드 / 없음 (이전/다음 가사 표시가 켜져 있으면 비활성화)
  • 미리보기 글자 크기 비율 / 투명도 이전/다음 가사 표시가 켜져 있을 때, 위·아래에 작게 보이는 미리보기 줄의 크기 비율(%)과 투명도(%)를 별도로 조절합니다
  • 텍스트 그림자 글자에 부드러운 그림자(drop shadow) 적용 여부
  • 외곽선 굵기 글자에 단단한 외곽선(stroke)을 추가 — 투명 배경에서 가독성을 높일 때 유용
  • 정렬 방향 가사창 및 텍스트 정렬: 왼쪽 / 가운데 / 오른쪽
  • 페이지 고정(Pin) 가사창을 페이지 좌표에 고정 — 스크롤해도 위치 유지. 가사창에 마우스를 올리면 나타나는 📌 핀 버튼으로도 토글 가능
  • 핀 색상 핀 아이콘 색상 설정 (배경색에 따라 조절)
  • 영역 선택 🎯 화면을 자유롭게 드래그하거나 특정 영역(예: 유튜브 플레이어)에 가사창 위치 지정
  • 사이트별 디자인 '현재 사이트'를 선택하고 저장하면 해당 사이트에만 다른 디자인을 적용할 수 있음
2 SRT 자막 작성 규칙
✅ 줄 구분 방법

한 자막 블록 안에서 줄을 나누는 방법은 두 가지입니다:

  • 텍스트 줄 내에서 \N (백슬래시 + 대문자 N) 사용
  • SRT 파일에서 텍스트를 실제 여러 줄로 작성
⚠️ 주의: \n 소문자는 줄 구분자로 인식되지 않습니다. 반드시 대문자 \N을 사용하세요.
1
00:00:10,000 --> 00:00:15,000
첫 번째 줄\N두 번째 줄

또는 실제 줄바꿈으로도 동일하게 동작:

2
00:00:16,000 --> 00:00:20,000
첫 번째 줄
두 번째 줄
✅ 1줄 / 2줄 / 3줄 자동 스타일

\N으로 구분된 줄 수에 따라 스타일이 자동으로 적용됩니다:

  • 1줄: 번역 스타일(크게·흰색)로 단독 표시
  • 2줄: 첫째 줄 = 원문, 둘째 줄 = 번역(크게)
  • 3줄: 첫째 줄 = 원문, 둘째 줄 = 발음, 셋째 줄 = 번역(크게)
1
00:00:05,000 --> 00:00:10,000
君の心へ続く長い一本道は\N키미노 코코로에 츠즈쿠 나가이 잇폰미치와\N너의 마음으로 이어지는 기나긴 외길은

3 플로팅 리모컨
🔘 활성화 방법

팝업 헤더의 리모컨 ON 버튼을 누르면 현재 사이트에서 리모컨이 표시됩니다. 사이트마다 개별 설정이 저장되므로, 유튜브는 ON·다른 사이트는 OFF 식으로 각각 관리할 수 있습니다.

🎮 버튼 구성
  • ⋮⋮ 드래그 핸들 드래그로 화면 위 임의 위치로 이동. 위치는 자동 저장됨
  • 🎵 라이브러리 저장된 가사 목록 패널 열기. 검색 후 클릭하면 즉시 로드 및 재생
  • 📜 타임라인 현재 가사의 전체 타임라인 패널 열기. 클릭 시 해당 위치로 이동
  • 🎯 영역 선택 오버레이가 표시될 영역을 마우스로 지정
  • ▶ / ■ 재생·일시정지 / 정지
  • − / 0.0s / + 싱크 ±0.1초 조절. 가운데 값을 클릭하면 0으로 초기화
  • 🔗 영상 동기화 등록 지금 화면에 보이는 가사 위치와 영상의 현재 재생 위치를 고정 등록 (유튜브/치지직 VOD 전용). 등록해두면 같은 영상을 다시 방문해도 자동으로 정확한 시점에 가사가 복원되고, 한 영상에 여러 곡을 시간대별로 등록할 수 있습니다
  • 🔒 / 🔓 동기화 임시 해제 자동 동기화가 어긋났을 때 잠시 연동을 끊고(🔓) −/+ 버튼이나 타임라인으로 미세조정한 뒤, 🔗 버튼을 다시 눌러 재등록(🔒로 복귀)할 수 있습니다
  • › 최소화 리모컨을 핸들만 남기고 최소화 / 다시 클릭으로 복원

  • D열에 1을 입력하면 Auto Only — 목록·검색에는 숨겨지고 자동 감지로만 로드됨 (SoundCloud 등)
  • E열은 리모컨의 🔗(영상-가사 싱크 동기화) 버튼으로 등록한 영상 정보 — URL|오프셋(ms) 형식이며, 한 곡이 여러 영상에 등록될 수 있으면 ;(세미콜론)으로 구분해 여러 개를 한 칸에 나열함 (예: url1|1000; url2|-500). 비워두면 기존 등록 내용이 그대로 유지됨.
💡 팁: 라이브러리 탭의 시트 양식 복사 버튼을 누르면 현재 저장된 모든 가사가 A~E 5열 형식으로 클립보드에 복사됩니다. 시트에 Ctrl+V만 하면 됩니다.
⬇️ 시트에서 가져오기
  1. 구글 시트 우측 상단 [공유] → 일반 액세스를 "링크가 있는 모든 사용자"로 변경 후 URL 복사
  2. 설정(⚙) 탭의 구글 시트 연동 URL 입력창에 붙여넣기 후 저장
  3. 라이브러리(📁) 탭에서 시트 연동 버튼 클릭
💡 병합 방식: 가져오기는 전체 덮어쓰기가 아닌 병합입니다. 동일 곡(아티스트+제목 기준)은 업데이트되고, 새 곡은 추가됩니다. 기존 데이터는 유지됩니다.
💡 자동 연동: 라이브러리가 비어있고 시트 URL이 저장돼 있으면, 팝업을 열거나 리모컨의 라이브러리 패널을 처음 열 때 자동으로 시트 데이터를 가져옵니다.
5 데이터 백업 및 복원 (JSON)
💾 라이브러리 데이터
  • 내보내기: 저장된 모든 가사 목록을 .json 파일로 저장
  • 가져오기: .json 파일을 불러와 현재 라이브러리에 병합 (동일 곡 업데이트 + 새 곡 추가, 전체 덮어쓰기 아님)
⚙️ 일반 설정 데이터
  • 내보내기: 구글 시트 URL, 리모컨 버튼 구성 등 동작·연동 관련 설정을 .json으로 저장
  • 가져오기: 저장해둔 일반 설정 파일을 불러오면 해당 설정이 즉시 복원됨
  • 기본값 복원: 일반 설정을 초기 기본값으로 리셋

  • 내보내기: 폰트·색상·애니메이션 등 시각 설정 + 사이트별 디자인 덮어쓰기 + 가사창 위치 정보를 .json으로 저장
  • 가져오기: 저장해둔 디자인 파일을 불러오면 모든 디자인 설정이 즉시 복원됨
  • 기본값 복원: 디자인 설정을 초기 기본값으로 리셋 (사이트별 덮어쓰기도 함께 초기화)

1 Basic Usage
🔘 Popup Header — Remote ON/OFF

Click the extension icon to open the popup. At the top you'll find a Remote ON / Remote OFF button. This is a per-site toggle — when ON, the floating remote control appears on the current page. Each website remembers its own setting independently.

🎵 Player Tab
  • Load SRT: Click a track in the library, or drag & drop a .srt file onto the player area
  • Play / Pause (▶ / ⏸): Control lyrics playback
  • Stop (■): Stop playback and reset position to the beginning
  • Progress bar: Click anywhere on the bar to jump to that position
  • Timeline: After loading an SRT, a scrollable list of all lyrics appears at the bottom of the Player tab. Click any entry to jump to that lyric.
  • Sync adjust: Fine-tune timing when lyrics are out of sync with the video
    ← → ±100ms  |  Shift+←→ ±500ms  |  R or the ↺ button to reset
📁 Library Tab
  • Add files: Drag & drop one or more .srt files onto the tab to register them all at once
  • Search: Search by number, artist, title, or keyword. Matching results show a chip indicating which field matched
  • Sheet Sync: Import lyrics from the configured Google Sheet URL and merge into the library
  • Copy Sheet Format: Copies all saved lyrics to the clipboard as a 3-column TSV (Name / SRT / Keywords) ready to paste into Google Sheets
  • Clear All: Delete all saved lyrics (confirmation dialog shown)
  • Manage Info: Opens a new tab for bulk-viewing/editing metadata — title, artist (original/Korean), number, keywords, Auto Only, video syncs — for every item at once
  • Edit Subtitles: The ✎ button on each item in the Manage Info page opens a popup editor for bulk timing shifts, per-line (original/pronunciation/translation) edits, and copy/paste of subtitle text
⚙️ Settings Tab (General)
  • End Notice Show a 🎵 Finish 🎵 notification after the last lyric ends
  • Progress Bar Show a thin progress bar at the bottom of the browser viewport
  • Auto-Detect Song Click "Manage" to open a window where you can toggle SoundCloud / YouTube / YouTube Music individually. Automatically detects the playing track and syncs lyrics
  • Remote Buttons Choose which buttons appear on the floating remote (Library / Timeline / Area / Play-Stop / Sync). The URL Sync buttons (🔗/🔓) can also be shown or hidden independently
  • Display Language How artist/title names are shown in library and player: Original+Korean / Original first / Korean first
  • Sheet URL Google Spreadsheet URL used for sheet sync
  • Show Previous / Next Lyric Shows the previous and next lines, smaller, right above and below the current line — like a karaoke screen
🎨 Design Tab
  • Original / Pron / English Show/hide, font size, and color for each lyric line independently
  • Background Background color, opacity (0–100%), and blur intensity
  • Animation Fade / Slide / None (disabled when Show Previous/Next Lyric is on)
  • Preview Font Scale / Opacity When Show Previous/Next Lyric is on, independently adjust the size ratio (%) and opacity (%) of the smaller preview lines shown above/below the current line
  • Text Shadow Toggle a soft drop-shadow on lyric text
  • Outline Width Add a solid stroke outline to the text — useful for readability on a transparent background
  • Alignment Lyric box and text alignment: Left / Center / Right
  • Pin to Page Anchors the lyric box to page coordinates — it stays in place when you scroll. Also toggled by the 📌 pin button that appears when hovering over the lyric box
  • Pin Color Color of the pin icon (adjust to match your background)
  • Select Area 🎯 Drag freely on screen or snap lyrics to a specific region (e.g., the video player)
  • Per-site Design Select "Current Site" and save to apply a different design on that site only
2 SRT File Format
✅ How to separate lines

There are two ways to split lines within a single subtitle block:

  • Use \N (backslash + uppercase N) within the text
  • Write the text on separate lines in the SRT file
⚠️ Note: Lowercase \n is NOT recognized as a line separator. Always use uppercase \N.
1
00:00:10,000 --> 00:00:15,000
First line\NSecond line

Or using actual line breaks in the file (same result):

2
00:00:16,000 --> 00:00:20,000
First line
Second line
✅ 1-line / 2-line / 3-line auto styling

The number of \N-separated lines determines which style is applied:

  • 1 line: Displayed in main style (large, white)
  • 2 lines: Line 1 = Original, Line 2 = Translation (large)
  • 3 lines: Line 1 = Original, Line 2 = Pronunciation, Line 3 = Translation (large)

1 基本的な使い方
🔘 ポップアップヘッダー — リモコン ON/OFF

拡張機能のアイコンをクリックするとポップアップが開き、上部に リモコン ON / リモコン OFF ボタンがあります。これはサイトごとの設定で、ON にすると現在のページ上にフローティングリモコンが表示されます。サイトごとに個別に記憶されます。

🎵 プレイヤータブ
  • SRT 読み込み: ライブラリから曲をクリックするか、プレイヤー領域に .srt ファイルをドラッグ＆ドロップ
  • 再生 / 一時停止 (▶ / ⏸): 歌詞の進行を制御
  • 停止 (■): 再生を完全に止め、先頭に戻る
  • プログレスバー: クリックするとその位置に移動
  • タイムライン: SRT 読み込み後、プレイヤータブ下部に歌詞一覧が表示されます。各項目をクリックするとその位置に移動します。
  • 同期調整: 映像と歌詞のタイミングがずれているときに調整
    ← → ±100ms  |  Shift+←→ ±500ms  |  R または UI の ↺ ボタンでリセット
📁 ライブラリタブ
  • ファイル追加: タブ内に .srt ファイルをドラッグ＆ドロップで複数ファイルを一括登録

  • シート同期: 設定に保存したGoogleシートURLからデータを取得してライブラリにマージ
  • シート形式コピー: 保存済みの全歌詞を3列TSV形式（名前 / SRT / キーワード）でクリップボードにコピー
  • 一括削除: 保存された全歌詞を削除（確認ダイアログあり）
  • 情報管理: タイトル・アーティスト（原文/韓国語）・番号・キーワード・Auto Only・動画シンクなどのメタデータを一括で確認・編集できる別ページを新しいタブで開きます
  • 字幕編集: 情報管理ページの各項目の✎ボタンから、タイミングの一括調整、行単位（原文/読み仮名/翻訳）の編集、テキストのコピー&ペーストができる編集画面をポップアップで開きます
⚙️ 設定タブ（一般）
  • 終了通知 最後の歌詞が終わった後に 🎵 Finish 🎵 通知を表示
  • プログレスバー ブラウザ下部に再生位置バーを表示するか
  • 楽曲自動検出 「管理」ボタンを押すと開くウィンドウで、SoundCloud / YouTube / YouTube Musicを個別にON/OFFできます。再生中の曲を自動検出して歌詞を読み込み・同期します
  • リモコンボタン フローティングリモコンに表示するボタンを個別選択（ライブラリ・タイムライン・エリア・再生停止・同期調整）。URL同期ボタン（🔗/🔓）の表示有無も別途ON/OFFできます
  • 表示言語 ライブラリ・プレイヤーでのアーティスト/タイトル表示方式: 原文+韓国語 / 原文優先 / 韓国語優先
  • シートURL シート同期に使用するGoogleスプレッドシートのURL
  • 前の歌詞を表示 / 次の歌詞を表示 現在の行の上下に、前後の歌詞を小さく一緒に表示します（カラオケ画面のように）
🎨 デザインタブ
  • 原文 / 読み仮名 / 日本語 各行の表示有無・フォントサイズ・色を個別設定
  • 背景 背景色・透明度(0〜100%)・ぼかし強度
  • アニメーション フェード / スライド / なし（前/次の歌詞表示がオンの場合は無効化）
  • プレビュー文字サイズ比率 / 透明度 前/次の歌詞表示がオンのとき、現在の行の上下に小さく表示されるプレビュー行のサイズ比率(%)と透明度(%)を別々に調整します
  • テキストドロップシャドウ 歌詞テキストに柔らかい影を付けるかどうか
  • 縁取りの太さ テキストに単色の縁取り（ストローク）を追加 — 透明背景での視認性向上に有用
  • 整列方向 歌詞ボックスおよびテキストの整列: 左 / 中央 / 右

  • エリア選択 🎯 画面上を自由にドラッグ、または特定領域（例: 動画プレイヤー）に歌詞ボックスを配置

2 SRT ファイルの書き方
✅ 行の区切り方

1つの字幕ブロック内で行を分ける方法は2つあります：

  • テキスト中に \N（バックスラッシュ + 大文字N）を使用
  • SRTファイル内でテキストを実際に複数行で記述
⚠️ 注意: 小文字の \n は行区切りとして認識されません。必ず大文字 \N を使用してください。
1
00:00:10,000 --> 00:00:15,000
1行目\N2行目

または実際の改行でも同じ結果になります：

2
00:00:16,000 --> 00:00:20,000
1行目
2行目
✅ 1行 / 2行 / 3行 の自動スタイル

\N で区切られた行数に応じてスタイルが自動的に適用されます：

  • 1行: メインスタイル（大きく・白色）で単独表示
  • 2行: 1行目 = 英語歌詞、2行目 = 日本語訳（大きく）
  • 3行: 1行目 = 英語歌詞、2行目 = 読み仮名、3行目 = 日本語訳（大きく）

3 フローティングリモコン
🔘 有効化の方法

ポップアップヘッダーの リモコン ON ボタンを押すと、現在のサイトでリモコンが表示されます。サイトごとに設定が保存されるため、YouTubeはON・他のサイトはOFFのように個別管理できます。

🎮 ボタン構成
  • ⋮⋮ ドラッグ ドラッグして画面上の任意の位置に移動。位置は自動保存
  • 🎵 ライブラリ 保存済み歌詞の一覧パネルを開く。検索してクリックするとすぐに読み込み・再生
  • 📜 タイムライン 現在の歌詞の全タイムラインパネルを開く。クリックでその位置に移動
  • 🎯 エリア選択 オーバーレイを表示する領域をマウスで指定
  • ▶ / ■ 再生・一時停止 / 停止
  • − / 0.0s / + 同期を±0.1秒調整。中央の値をクリックすると0にリセット
  • 🔗 動画-歌詞シンク同期登録 現在表示中の歌詞位置と動画の再生位置を固定登録（YouTube/CHZZK VODのみ対応）。登録しておくと同じ動画を再度開いても自動的に正確なタイミングで歌詞が復元され、1つの動画に複数の曲を時間帯別に登録できます
  • 🔒 / 🔓 シンクを一時解除 自動同期がずれた場合、一時的に連動を解除（🔓）してから−/+ボタンやタイムラインで微調整し、🔗ボタンを再度押して再登録（🔒に戻る）できます
  • › 最小化 リモコンをドラッグハンドルのみに縮小。再クリックで復元

⬇️ シートからインポート
  1. Googleシート右上の [共有] → 一般アクセスを 「リンクを知っている全員」 に変更してURLをコピー
  2. 設定(⚙)タブ の Googleシート同期 URL欄に貼り付けて保存
  3. ライブラリ(📁)タブ で シート同期 ボタンをクリック
💡 マージ方式: インポートは完全上書きではなく マージ です。同じ曲（アーティスト+タイトル）は更新され、新しい曲は追加されます。既存データは保持されます。
💡 自動同期: ライブラリが空でシートURLが保存されている場合、ポップアップを開いたときやリモコンのライブラリパネルを初めて開いたときに自動的にシートデータを取得します。

💾 ライブラリデータ
  • エクスポート: 保存済みの全歌詞を .json ファイルとして保存
