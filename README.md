# Lyrics Overlay

SRT 자막 파일을 기반으로 영상/오디오 위에 싱크 가사를 오버레이로 표시하는 Chrome 확장
프로그램입니다. 원문/발음/번역을 한 화면에 함께 보여주고, 유튜브·치지직·SoundCloud 등에서
재생 위치에 맞춰 자동으로 따라갑니다.

## 주요 기능

- **SRT 가사 오버레이**: 원문/발음/해석 3줄 표시, 폰트·색상·정렬·애니메이션 등 디자인 커스터마이즈
- **플로팅 리모컨**: 페이지 위에 떠 있는 컨트롤러로 재생/정지, 싱크 미세조정, 곡 목록, 타임라인 탐색
- **자동 감지 동기화**: SoundCloud / YouTube / YouTube Music에서 재생 중인 곡을 자동으로 인식해 가사 매칭
- **영상-가사 URL 동기화**: YouTube/치지직(VOD) 영상의 특정 시점과 가사 위치를 고정 등록해, 같은 영상을 다시 봐도 자동으로 정확한 싱크가 복원됨(영상 1개에 여러 곡 등록 가능)
- **라이브러리 관리**: 곡 목록을 로컬에 저장하고, Google Sheet와 동기화해 여러 기기에서 공유 가능
- **자막 편집기 / 정보 관리 페이지**: 자막 타이밍/본문 직접 수정, 제목·가수·키워드 등 메타데이터 관리

## 설치

[Chrome 웹스토어](https://chromewebstore.google.com/detail/lyrics-overlay/lndbfkednopjjifcgmfoeggjkpimglfo)에서
설치하세요.

<details>
<summary>개발자 모드로 직접 설치하기</summary>

1. 이 저장소를 클론하거나 ZIP으로 다운로드합니다.
2. Chrome에서 `chrome://extensions` 접속.
3. 우측 상단의 "개발자 모드"를 켭니다.
4. "압축해제된 확장 프로그램을 로드" 클릭 후, 이 프로젝트 폴더(`manifest.json`이 있는
   루트)를 선택합니다.

</details>

## 사용법

확장 아이콘을 클릭해 팝업을 열고 SRT 파일을 추가한 뒤, 영상/오디오 페이지에서 플로팅
리모컨을 켜서 재생하면 가사가 오버레이로 표시됩니다. 자세한 사용법은 확장 내 도움말
페이지(팝업 → 도움말)를 참고하세요.
