🎧 내 브라우저 안의 완벽한 노래방! '싱크 가사 오버레이 (Lyrics Overlay)'

유튜브, 치지직, 트위치, 사운드클라우드 등 어떤 스트리밍 환경이든 내가 원하는 가사를 화면 위에 예쁘게 띄워보세요! 복잡한 영상 편집이나 무거운 프로그램 없이, 브라우저 화면 위에 바로 가사(자막)를 동기화하여 띄워주는 강력하고 가벼운 확장프로그램입니다.


✨ [핵심 기능 가이드]

1. 🎵 SoundCloud 자동 감지 & 자동 동기화
SoundCloud에서 곡을 재생하면 라이브러리에 등록된 가사를 자동으로 찾아 재생 위치에 맞춰 싱크합니다. 타임라인을 이동하거나 곡을 반복해도 가사가 끊김 없이 따라옵니다.

2. 🎯 자유로운 오버레이 & 영역 지정
화면 어디든 원하는 위치에 가사를 배치하세요. '영역 지정 🎯' 기능으로 영상 플레이어 크기에 딱 맞게 고정하거나, '핀 고정' 기능으로 페이지 좌표에 고정해 스크롤해도 위치가 유지되도록 할 수 있습니다.

3. 🕹️ 강력한 '플로팅 리모컨'
팝업창을 열지 않고도 화면 위의 미니 리모컨으로 모든 것을 제어하세요!
- 상시 표시 (사이트별 설정): 자주 가는 사이트에서 '리모컨 ON'을 켜두면 방문 시 자동으로 나타납니다.
- 플랫폼 맞춤 테마: 접속한 사이트에 맞춰 색상이 변경됩니다 — 치지직(초록), 유튜브(빨강), 트위치(보라), 사운드클라우드(주황).
- 스마트 패널: 리모컨 하나로 가사 목록 검색·타임라인 확인을 즉시 수행하며, 패널이 위아래로 똑똑하게 펼쳐집니다.
- 버튼 커스텀: 표시할 버튼(목록·타임라인·영역 선택·재생/정지·싱크)을 설정에서 골라 구성할 수 있습니다.
- 최소화 모드: 클릭 한 번으로 작게 접어 영상 시청을 방해하지 않습니다.

4. 📁 초간단 라이브러리 & 스마트 검색
SRT 자막 파일을 드래그 앤 드롭으로 간편하게 추가하세요. 수많은 가사를 저장해 두고 검색으로 1초 만에 불러올 수 있습니다. 검색 시 가수·제목·키워드 중 어디에서 일치했는지 칩으로 표시되고, 일치한 글자는 강조색으로 하이라이트됩니다.

5. ☁️ 구글 스프레드시트 연동 (클라우드 동기화)
주소만 입력하면 모든 가사 데이터가 라이브러리로 즉시 동기화됩니다. 시트 D열을 'Auto Only'로 설정하면 해당 가사는 SoundCloud 자동 감지 전용으로 등록되어 일반 목록에는 표시되지 않습니다. 라이브러리가 비어있을 경우 자동으로 데이터를 불러와 처음 설치 후 바로 사용할 수 있습니다.

6. 🎨 나만의 커스텀 디자인
원문 / 발음 / 번역 3줄 자동 분리 표시를 지원합니다. 글꼴, 색상, 배경 투명도, 블러, 텍스트 외곽선, 애니메이션, 가사창 정렬 방향(좌/중/우)까지 내 입맛대로 스타일링하세요. 사이트별로 다른 디자인을 적용하는 개별 설정도 지원합니다.

7. ⏱️ 섬세한 싱크 조절
키보드 방향키(←, →)로 0.1초, Shift+방향키로 0.5초 단위 조절이 가능하며, 리모컨 버튼으로도 미세 조정할 수 있습니다.

8. 🔼🔽 이전/다음 가사 표시
현재 가사 위·아래에 이전·다음 줄을 작게 함께 보여줍니다. 노래방처럼 다음 가사를 미리 확인할 수 있고, 줄 수가 바뀌거나 빈 구간이 와도 가사창 높이가 고정되어 출렁이지 않습니다.

9. 📜 추가 편의 기능
- 자동 스크롤: 곡 제목·가수명이 길면 전광판처럼 부드럽게 스크롤됩니다.
- 곡 종료 알림: 가사가 모두 끝나면 화면에 🎵 Finish 🎵 문구를 표시합니다. (설정에서 켜고 끄기 가능)
- 진행 바: 브라우저 하단에 재생 위치 바를 표시합니다.
- 백업 / 복원: 일반 설정과 디자인 설정을 각각 독립적으로 JSON 파일로 내보내고 가져올 수 있습니다.

🔒 [개인정보 및 보안]
'싱크 가사 오버레이'는 오직 사용자의 브라우저 로컬 환경(chrome.storage)에서만 동작합니다. 사용자가 직접 추가한 데이터 외에 어떠한 개인정보도 외부 서버로 수집하거나 전송하지 않아 100% 안전합니다. (구글 시트 연동 시에는 사용자가 입력한 시트 주소에만 접근합니다.)
